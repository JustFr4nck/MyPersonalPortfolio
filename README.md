# 🌐 My Personal Portfolio

Designed to be fast, responsive, and easy to customize.

## ✨ Key Features

- **Responsive Design** → Optimized for desktop, tablet, and mobile  
- **Smooth Animations** → Transitions and micro-interactions for a modern feel  
- **Customizable Theme** → Change colors and fonts in `tailwind.config.js`  
- **Modular Sections** → Home, About, Projects, Contact  
- **SVG Icon Integration** → Manage SVGs as Vue components  
- **Fast Loading** → Thanks to Vite and asset optimization  
- **Simple State Management** → Using Pinia for reactive components
