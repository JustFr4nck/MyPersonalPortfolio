# 🌐 My Personal Portfolio

Un portfolio personale moderno sviluppato con **Vue 3**, **Tailwind CSS**, **DaisyUI** e **Pinia**.  
Progettato per essere veloce, responsive e facile da personalizzare.

## 🛠️ Tech Stack

**Frontend Framework:**  
- [Vue 3](https://vuejs.org/) – Framework JavaScript progressivo

**Styling & UI:**  
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework  
- [DaisyUI](https://daisyui.com/) – Componenti pre-stilizzati per Tailwind  
- [Unplugin Icons](https://github.com/antfu/unplugin-icons) – Icone SVG come componenti Vue

**State Management & Routing:**  
- [Pinia](https://pinia.vuejs.org/) – Gestione stato  
- [Vue Router](https://router.vuejs.org/) – Routing client-side

**Build & Tools:**  
- [Vite](https://vitejs.dev/) – Bundler ultrarapido  
- [TypeScript](https://www.typescriptlang.org/) – Tipizzazione statica  
- [ESLint](https://eslint.org/) & [Prettier](https://prettier.io/) – Linting e formattazione

---

## ✨ Key Features

- **Responsive Design** → Ottimizzato per desktop, tablet e mobile  
- **Animazioni fluide** → Transizioni e micro-interazioni  
- **Tema personalizzabile** → Modifica colori e font in `tailwind.config.js`  
- **Sezioni modulari** → Home, About, Projects, Contact  
- **Integrazione icone SVG** → Gestite come componenti Vue  
- **Caricamento veloce** → Grazie a Vite e ottimizzazione asset  
- **Gestione stato semplice** → Con Pinia per componenti reattivi
